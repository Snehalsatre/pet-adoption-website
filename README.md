# 🐾 Pet Adoption Website

A simple **Pet Adoption Website** created using **HTML, CSS, Bootstrap, and JavaScript**.
This project provides a platform where users can view pets available for adoption and learn more about them.

---

## 📌 Features

* 🐶 View pets available for adoption
* 🖼️ Pet gallery with images and information
* 📋 Adoption form
* 📱 WhatsApp button to know more about pets
* 📞 Contact page
* 🎨 Responsive design using Bootstrap

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* Bootstrap 5
* JavaScript
* Font Awesome Icons
